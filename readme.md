# VPC AWS
Build VPC in AWS.

## Installation

Use the binary manager [terrafrom](https://www.terraform.io/downloads.html/) to install foobar.

```bash
mv terrafom /user/bin/terraform
```

## Usage

```python
terraform init
terraform validate
terrafom plan -out nameYourPlan
terraform apply nameYourPlan
'
```
